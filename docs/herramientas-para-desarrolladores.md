## httpie

>  Línea de comandos de HTPP moderna - alternativa a CURL amigable con el usuario con una una intituiva interfaz de usuario; soporte para JSON; resaltado de sintaxis, comando de tipo wget, extensiones, etc.
> https://github.com/jakubroztocil/httpie
